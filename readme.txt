== Installation ==

1. Ensure you have latest version of WooCommerce plugin installed (WooCommerce 2.0+)
2. Unzip and upload contents of the plugin to your `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. set API keys and configure settings (See Configuration)

== Configuration ==

1. Visit the `WooCommerce > Settings > Checkout` tab.
2. Click on *Uviba Payments* to edit the settings. If you do not see *Uviba Payments* in the list at the top of the screen make sure you have activated the plugin in the WordPress Plugin Manager.
3. Enable the Payment Method, name it `Credit Card / Debit Card / Internet Banking` (this will show up on the payment page your customer sees).
4. Add your API Keys (public live, private live, public test, private test keys).
5. Define *How much {Your store currency} is one dollar*. It is extremely important because Uviba payments will use this number to convert your currency to dollar and charge users.
6. Select `Redirect url` (URL you want Uviba Payments to redirect after payment).
7. Click Save.
